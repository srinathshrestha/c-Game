# 🏎️ Car Racing

A 2D top-down car racing game written in C using SDL2.

Dodge oncoming traffic, rack up points, and survive as the speed ramps up.

## Prerequisites

You need a C compiler and the SDL2 development libraries:

**macOS**
```bash
brew install sdl2 sdl2_image sdl2_ttf
```

## Build & Run

```bash
git clone https://github.com/srinathshrestha/c-Game.git
cd c-Game
make
./car_game
```

To rebuild from scratch:
```bash
make rebuild
```

## Controls

| Key | Action |
|---|---|
| `← →` or `J / L` | Steer left / right |
| `↑ ↓` or `W / S` | Move up / down |
| `Space` | Start / Restart |
| `P` | Pause / Unpause |
| `M` | Toggle music |
| `Esc` | Quit |
